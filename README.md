# Home Solar Portable Emergency Charger

## Projektbeschreibung / Project Description

**Deutsch**
Dieses Projekt beschreibt den Bau eines portablen Notstrom-Ladegeräts für Heimspeicher-Systeme (insbesondere E3/DC) unter Verwendung eines Generators. Da viele Heimspeicher-Systeme (wie E3/DC) ein direktes Laden der Batterie über einen Generator am AC-Eingang aufgrund von Frequenzinstabilitäten oft verweigern, wird hier ein anderer Ansatz gewählt: Der Generator speist ein leistungsstarkes DC-Netzteil (MeanWell CSP-3000-400), welches eine Gleichspannung erzeugt, die direkt in die PV-Eingänge (MPPT-Tracker) des Wechselrichters eingespeist wird. Das System "simuliert" somit eine Solaranlage.

**English**
This project describes the construction of a portable emergency charger for home battery systems (specifically E3/DC) using a generator. Since many home battery systems often refuse to charge directly from a generator via the AC input due to frequency instabilities, this project uses a different approach: The generator feeds a powerful DC power supply (MeanWell CSP-3000-400), which generates a DC voltage that is fed directly into the PV inputs (MPPT trackers) of the inverter. The system effectively "simulates" a solar array.

## Wichtiger Hinweis / Important Notice

**Deutsch**
Dies ist ein DIY-Projekt, das Arbeiten an Hochspannung (bis zu 400V DC und 230V AC) beinhaltet. Unsachgemäße Handhabung kann zu lebensgefährlichen Verletzungen oder Schäden an der Anlage führen. Der Nachbau geschieht auf eigene Gefahr. Es wird dringend empfohlen, Elektrofachkräfte hinzuzuziehen.

**English**
This is a DIY project involving high voltage (up to 400V DC and 230V AC). Improper handling can lead to life-threatening injuries or damage to the system. Replication is at your own risk. Consultation with qualified electricians is strongly recommended.
